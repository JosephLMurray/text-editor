# Text Editor

## Description

A PWA text editor that uses IndexedDB to store data locally.

## Table of Contents

- [Installation](#installation)
- [Usage](#usage)
- [Credits](#credits)
- [License](#license)

## Installation

npm install

## Usage

After installation, run 'npm run build' and then 'npm run start' to initialize the app locally.
